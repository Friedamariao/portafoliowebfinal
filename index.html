Parcial Dos

En el Segundo Parcial acerca CSS, se abordaron varios temas clave que son fundamentales para comprender y trabajar efectivamente con hojas de estilo en cascada. Algunos de los temas vistos fueron:

Modelo de caja: Se exploraron los conceptos relacionados con el modelo de caja en CSS, que incluyen el contenido, el padding, el borde y el margen. Comprender cómo estos elementos interactúan es esencial para controlar el diseño y la apariencia de los elementos en una página web.

Selectores: Son herramientas para aplicar estilos a elementos específicos en un documento HTML. Durante el parcial, se discutieron diferentes tipos de selectores, como selectores de tipo, de clase, de ID, de descendencia, de hijo directo, de atributo, entre otros. Estos selectores permiten dirigir y estilizar con precisión partes específicas de una página web.

Transformación y animación: La transformación y la animación en CSS ofrecen la capacidad de crear efectos visuales dinámicos y atractivos en una página web. Durante el parcial, se cubrieron propiedades como transform, que permite modificar la posición, tamaño y rotación de elementos, y animation, que permite crear animaciones personalizadas con transiciones suaves entre diferentes estados de un elemento.

Parcial Dos

Parcial Dos

En el Parcial 3 de JavaScript, profundizamos varios aspectos clave del lenguaje de programación JavaScript, centrándonos en la incorporación de bibliotecas, el uso de APIs, la estructura del código, los eventos, las funciones y la declaración de variables.

Uso de APIs: Aprendimos sobre las APIs disponibles en JavaScript y cómo acceder a ellas para interactuar con servicios externos y recursos en la web. Esto abarcó el uso de APIs para crear graficos, creacion de pdfs y acceder a la geolocalización del usuario.

Eventos: Exploramos los conceptos de eventos en JavaScript y cómo utilizarlos para responder a las acciones del usuario, como hacer clic en un botón y enviar un formulario. Aprendimos a asignar manejadores de eventos y a utilizar métodos como onCLick() y onSubmit().

Funciones y declaración de variables: Reforzamos los conceptos de funciones y variables en JavaScript. Aprendimos a declarar variables utilizando var, let y const, y a utilizar funciones para encapsular y reutilizar bloques de código. Cubrimos temas como la declaración de funciones, los parámetros y los valores de retorno, así como el alcance de las variables.
